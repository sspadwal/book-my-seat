# Cinema Seat Booking System

This project is a simple web application for booking cinema seats.

## What it does
- Users can create an account and log in.
- Authenticated users can view a grid of seats.
- Users can book available seats in real-time.

## Project Structure
- **client**: The frontend built with React and Vite.
- **server**: The backend built with Node.js, Express, and PostgreSQL.
